# Project-Management-Dashboard
A modern, responsive project management dashboard built with HTML, CSS, and JavaScript. This dashboard provides teams with a comprehensive overview of their projects, tasks, and team performance.
# Features

Dashboard Overview: Get a quick summary of active projects, completed tasks, team members, and productivity metrics

Task Management: Create, track, and manage tasks with priorities and due dates

Team Management: View team members with their roles and performance metrics

Analytics: Visualize project progress with interactive charts and graphs

Calendar Integration: View project timelines and deadlines

Settings: Customize your dashboard preferences and notifications

Responsive Design: Works seamlessly on desktop, tablet, and mobile devices

# Technologies Used

HTML5

CSS3 (Flexbox, Grid, Custom Properties)

JavaScript (ES6+)

Chart.js for data visualization

Font Awesome for icons

# Getting Started

## Prerequisites

A modern web browser (Chrome, Firefox, Safari, Edge)

Basic understanding of web development (for customization)

## Installation

Clone or download this repository

Navigate to the project directory

Open index.html in your web browser

## Usage

Dashboard: View your project statistics and recent tasks

Projects: Manage your active projects

Calendar: Check your project timeline and deadlines

Team: View team members and their responsibilities

Analytics: Analyze project performance with interactive charts

Settings: Customize your dashboard preferences

Customization
Adding New Tasks
Click the "New Task" button on the dashboard to create a new task. Fill in the task details in the modal form and submit.
